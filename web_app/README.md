# Училищна библиотека

React + TypeScript · Express + TypeScript + MySQL · JWT · Тъмна/Светла тема

## Изисквания

- Node.js 18+ → https://nodejs.org
- XAMPP → https://www.apachefriends.org

## Стартиране

Стартирайте MySQL от XAMPP, после в терминала:

```
npm install
npm run dev
```

Базата данни, таблиците и примерните данни се създават автоматично.
